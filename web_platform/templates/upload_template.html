模板設計說明

請在您的 Word 模板中，使用 {{ '{{欄位名}}' }} 的格式來標示要替換的欄位。

例如：

專案名稱：{{ '{{project_name}}' }}
公司名稱：{{ '{{company_name}}' }}
報價：{{ '{{price}}' }}
聯絡人：{{ '{{contacts}}' }}

系統會自動偵測這些佔位符，並在批次產生文件時替換成對應資料。
